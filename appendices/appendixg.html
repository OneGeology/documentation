Appendix G: WMS 1.1.1 GetCapabilities response from the BGS OneGeology exemplars service

Below is the GetCapabilities response returned by the BGS OneGeology service as configured by the MapServer map file shown in Appendix E. This response document may be obtained using the following GetCapabilities request like:

http://ogc.bgs.ac.uk/cgi-bin/exemplars/BGS_Bedrock_and_Superficial_Geology/ows?
 service=WMS&
 request=GetCapabilities&
 version=1.1.1&

Note you must specify the version here in this request because there is a higher WMS version available on the server.
